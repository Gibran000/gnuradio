    mkdir workarea-gnuradio
    cd workarea-gnuradio
    git clone --recursive https://github.com/gnuradio/gnuradio
    cd gnuradio
    git checkout v3.7.13.5 (Si no aparece solamente "HEAD" y el volk, inténtalo otra vez)

Instalación de UHD en ubuntu 16.04

Siguiendo los pasos de Ettus Researcher

[https://kb.ettus.com/Building_and_Installing_the_USRP_Open-Source_Toolchain_(UHD_and_GNU_Radio)_on_Linux]
:


    mkdir workarea-uhd
    cd workarea-uhd
    git clone https://github.com/EttusResearch/uhd
    git checkout v3.13.0.0
    cd host
    mkdir build
    cd build
    cmake ../
    make
    make test
    sudo make install
    sudo ldconfig
    export LD_LIBRARY_PATH=/usr/local/lib

Instalación del bloque gr-ieee802-11 en ubuntu 16.04



    Ir al repositorio github: https://github.com/bastibl/gr-ieee802-11 y descargar la versión maint3.7
    descomprimir y mover la carpeta hacia la ruta donde se instaló GNU Radio "workarea-gnuradio/gnuradio"
    entrar a ésta carpeta (gr-ieee802-11-maint.37), abrir la terminal y ejecutar los siguientes comandos:
    mkdir build
    cd build
    cmake ../
    make
    make test
    sudo make install
    sudo ldconfig
